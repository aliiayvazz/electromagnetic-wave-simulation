# Uniform Plane Wave Propagation Simulation (MATLAB)

This project provides a comprehensive computational analysis and simulation of **Electromagnetic Plane Wave Propagation** within lossy media. Developed as part of the Electromagnetic Field Theory curriculum, it models the physical behavior of waves using complex mathematical foundations.

## Simulation Capabilities
* **Lossy Media Analysis:** Calculates propagation constants including Attenuation ($\alpha$) and Phase ($\beta$) constants for diverse material properties.
* **Field Vector Modeling:** Visualizes the spatial and temporal evolution of Electric ($E$) and Magnetic ($H$) field vectors.
* **Impedance & Phase Calculations:** Computes intrinsic impedance and the phase difference between $E$ and $H$ fields.
* **Power Flow:** Models the Poynting vector to analyze energy density and power transmission.

## Technology & Tools
* **Language:** MATLAB (.m)
* **Mathematical Focus:** Maxwell's Equations, Complex Phasors, Wave Equations.

## Engineering Value for Defense & Telecommunications
Understanding how waves interact with different environments is the backbone of:
* **Radar Cross Section (RCS) Calculations:** How waves reflect and absorb.
* **Antenna Design:** Near-field and far-field wave behaviors.
* **Stealth Technology:** Wave attenuation in specialized coatings.
